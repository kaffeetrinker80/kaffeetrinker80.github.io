# kaffeetrinker80.github.io

Personal collection of practical web tools and helper projects.

This site contains several small HTML/JavaScript tools that I built mainly for everyday work, congregation tasks and event organization.

## Sections

- `/worktools/`
  Practical tools for work and water hygiene documentation  
  Example: flushing logs, temperature records, service reports

- `/theocratic/`
  Tools for congregation and convention organization  
  Example: microphone service, volunteer planning, counting tools

- `/private/`
  Personal experiments, drafts and internal helper tools

## Philosophy

Most tools are:

- simple single-file HTML tools
- running locally in the browser
- storing data locally as JSON
- designed to work without a server
- optimized for easy printing and mobile use

## Current Projects

- Microphone Service Tool
- Volunteer / Regional Convention helper tools
- Spülprotokoll / Arbeitsnachweis
- PTC Tool
- Various small planning and documentation helpers

## Notes

These tools are mainly built for personal use and may still change over time.

Some tools may require:
- importing/exporting JSON files
- running from GitHub Pages
- optional local webserver for advanced browser features
